Разрешённые HTML-теги и атрибуты

Вы можете использовать следующие теги для оформления профиля:

Блочные: p, div, h1, h2, h3, h4, h5, h6, blockquote, pre, hr, center
Строчные: span, a, strong, em, u, b, i, big, small, sub, sup, strike, s, del, ins
Списки: ul, ol, li
Таблицы: table, tr, td, th, thead, tbody
Медиа: img, figure, figcaption
Прочее: br, code, details, summary, font
Атрибуты: class, style, id, title, align, valign, colspan, rowspan, href, target, src, alt, width, height, color, size, face, border, cellpadding, cellspacing
Важно: JavaScript, iframe, embed, object, script, form, input не разрешены и будут удалены.

CSS можно использовать любой, но помните, что он будет применён только к вашему профилю. Не добавляйте скрипты — они будут вырезаны.
